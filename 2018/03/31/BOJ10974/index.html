[10974] 모든 순열

2018-03-31
BOJ

문제 : https://www.acmicpc.net/problem/10974

Key Point

잘 사용하지 않은 반복문을 사용해서 첫번째 순열을 찍고 다음 순열을 계속해서 찍어내면 된다.
do ~ while문을 사용하지 않으면 사전순으로 첫 번째에 위치한 순열을 찍어낼 수 없으므로 do ~ while문을 사용한다.
Explain
이전 순열과 모든 순열을 구하는 문제를 하나로 합치면 된다.
