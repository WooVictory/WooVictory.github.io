개요

Lang은 내국인과 외국인이 함께하는 글로벌 언어교환 & 모임 서비스

국내에 체류하고 있는 외국인의 수가 증가하고 글로벌 비즈니스 사회에서 외국어 능력의 중요도가 증가하고 있지만, 정작 우리는 소통을 위한 외국어 공부가 아닌 문법 위주의 공부를 하고 있습다.
가장 좋은 외국어 학습은 외국인과 실제로 언어 교환을 하는 것이지만, 내국인과 외국인의 상호교류에 대한 정보와 플랫폼이 부족한 현실에서 이를 해결하기 위해 언어 교환을 위한 모임을 개설하고 외국인과 Communication을 할 수 있는 장을 마련해주는 플랫폼이자 애플리케이션입니다.

개발

로그인

카카오톡과 연동을 하여 유저의 정보를 받아서 안드로이드에서 정보를 저장할 수 있는 방법인 SharedPreferencesService를 사용해서 id, email, name을 저장하고 sign() 함수를 호출해서 저장된 정보를 불러와 회원가입을 진행하고 있습니다.

sign() 함수에서 SharedPreferencesServiece를 통해서 저장된 정보를 불러와 회원가입을 진행하였습니다.

회원가입

회원가입은 4개의 Activity로 구성되어 있고 총 2단계로 구분하였습니다.
1단계 : 약관동의와 개인정보 취급 방침
2단계 : 사용자의 언어, 배우고 싶은 언어 선택

1단계
Text 파일로 작성된 이용약관과 개인정보 취급 방침은 InputStreamReader를 통해 읽고 View에 뿌려주었습니다. 그리고 TextView의 속성인 movementMethod중 ScrollingMovementMethod()를 사용하여 스크롤이 가능하게끔 구현하였습니다.

2단계
사용자가 사용하는 언어와 배우고 싶은 언어를 선택할 수 있도록 Custom Dialog를 만들었고, 각 항목은 RadioGroup과 RadioButton을 이용하여 화면을 구성하였습니다.
Dialog를 Layout으로 작성하고 LayoutInfalter를 통해서 inflate 하여 사용하였습니다.

Main 화면

Main 화면은 5개의 Tab으로 구성되어 있습니다. TabLayout을 사용하여 하단 Tab을 구현하였고, Tab과 상단 Toolbar를 제외한 부분은 Fragment를 사용하여 화면을 분할하였습니다.
그리고 상단의 필터기능을 추가하여, 전체, 언어교환, 파티 3가지로 필터링해서 볼 수 있도록 구현하였고
Activity에서 Fragment로 모임 목적에 대한 type 값을 전달해야 하므로 Fragment()객체를 생성하고 bundle 객체를 이용하여 type 값을 넘기고 재통신을 진행하였습니다.

모임 목적을 type 값으로 구분
전체 : 100, 언어교환 : 101, 파티 : 102

Filter 구현 - bundle을 이용한 값의 전달

bundle 객체를 통해 값을 전달 받아서 type에 따른 재통신 진행

모임 상세 화면

모임 상세 화면에서는 모임의 상세 정보를 볼 수 있도록 구현하였습니다.
Activity에 들어가는 내용이 많이 ScrollView를 이용하였고, 서버로부터 모임의 정보를 받아와서 Activity에 뿌려주었습니다. 구글 Map API를 사용하여 서버로부터 위도/경도 정보를 받아와서 주소로 변환하여 지도에 표시되도록 구현하였습니다.

RatingBar를 Custom해서 사용하여 모임에 대한 후기를 작성할 수 있도록 구현하였습니다.
